Version 1.1.6 Released, 4th of January, 2010

  • Completely revamped the web site to use the Django template language rather than my custom code which is more powerful and should require less work in the future.
  • Jumping Tower's bullets can now jump to a sprite that's already been hit by that bullet rather than being restricted to ones that it hasn't hit yet, making it stronger.
  • A few other minor tweaks and bug fixes.

Version 1.1.5 Released, 7th of September, 2009

  • Added a high scores table (not in the applet version).
  • Made weaken tower gain damage and kills that result from its effect.
  • Changed the way homing tower's bullets work, try it and see.
  • Improved the rate that the length of the slow of slow tower increases by.
  • Simplified the drawing routine somewhat, which now drops frames if the drawing starts to lag.
  • A few other minor bug fixes.

Version 1.1.4 Released, 11th of August, 2009

  • Changed the stats of a few towers to make it more balanced.
  • Changed how some of the buttons appear to make them more uniform.
  • You are now able to see the licence from the title screen.
  • Reduced the memory usage by 10-20%
  • A few other bug fixes and refactorings.

Version 1.1.3 Released, 13th of July, 2009

  • Made the game skinnable, and added a new default skin (for towers and the bullet). Some of the pictures look a bit weird, because I can't draw.
  • A few other bug fixes, performance tweaks and refactorings.

Version 1.1.2 Released, 8th of July, 2009

  • Sprites now change colours when they are slowed/weakened/poisoned.
  • Right-clicking cycles through the speeds in reverse, and added +/- as keyboard shortcuts for that.
  • The text on the end-level display is now antialiased so looks nicer.
  • Revamped the way the multithreading works, is much simpler now.
  • Fixed a few bugs, made a few performance improvements and some refactoring.

Version 1.1.1 Released, 10th of June 2009

  • Improved performance of the new maps significantly.
  • Fixed the ridiculously high memory use of the previous version when the new maps were used.
  • Fixed a couple of other minor bugs.

Version 1.1.0 Released, 8th of June 2009

  • Added a lot more detail to the manual, most aspects of the game should be covered there now.
  • Added damage bonuses for a sprite being hit by different tower types. More detail in the manual.
  • Changed the format of the map description files to be a nice XML format rather than the nasty one I was using previously.
  • Added 3 new maps.
  • Lots of under the hood changes and refactoring.

Version 1.0.1 Released, 19th of February 2009

  • Incomplete manual added, currently lists all the towers.
  • Fast forward button letting you choose between 1x, 2x, and 5x normal speed.

Version 1.0.0 Released, 4th of February 2009

  • Initial numbered release
  • Fully playable
  • 18 available towers (including the ghost)
  • 3 available maps
