Version 1.1.1 Released, 10th of June 2009
  • Improved performance of the new maps significantly.
  • Fixed the ridiculously high memory use of the previous version when the new maps were used.
  • Fixed a couple of other minor bugs.

Version 1.1.0 Released, 8th of June 2009
  • Added a lot more detail to the manual, most aspects of the game should be covered there now.
  • Added damage bonuses for a sprite being hit by different tower types. More detail in the manual.
  • Changed the format of the map description files to be a nice XML format rather than the nasty one I was using previously.
  • Added 3 new maps.
  • Lots of under the hood changes and refactoring.

  • Incomplete manual added, currently lists all the towers.
  • Fast forward button letting you choose between 1x, 2x, and 5x normal speed.
